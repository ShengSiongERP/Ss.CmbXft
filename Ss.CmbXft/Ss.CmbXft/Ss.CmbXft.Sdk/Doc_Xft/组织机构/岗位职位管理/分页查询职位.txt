分页查询职位
更新时间: 2025-04-17 07:30:01
接口调用地址
生产环境地址：https://api.cmbchina.com/hrm/hrm2/organization-management/openapi/job/query/page
测试环境地址：https://api.cmburl.cn:8065/xfthrm/hrm2uat/organization-management/openapi/job/query/page
接口描述
分页查询职位
调用方式
METHOD: POST
CONTENT-TYPE: application/json
接口输入参数

请求体
仅必填全部展开
参数	必填	类型	最大长度	名称	示例值	字段说明
					
	否	ARRAY	1000	流水号列表	["0000**01"]	每项最大长度为10
sequenceNumbers
	否	STRING	150	职位名称	高级*理	支持模糊查询
jobName
	否	STRING	8	职位编号	J**01	支持模糊查询
codeNumber
	是	LONG	-	当前页数	1	当前页数，起始值1
currentPage
	是	LONG	-	每页大小	50	每页大小，默认值10，最大支持1000
pageSize

响应体

	否	OBJECT	-			-

	是	LONG	-	当前页数	1	当前页数，起始值1
currentPage
	是	LONG	-	每页大小	50	每页大小，默认值10
pageSize
	是	LONG	-	数据总条数	1	-
totalSize
	否	ARRAY	-	查询数据列表		-
records
	否	STRING	10	职位流水号	0000**01	-
sequenceNumber
	否	STRING	8	职位编号	J0**10	-
codeNumber
	否	STRING	150	职位名称	高级*理	-

errorMsg

错误码说明
除HTTP返回码外，具体的业务异常错误码通常指的是返回报文中错误码字段的内容，常见通用错误码及解决方案请参考通用错误码。
错误码	错误信息
XFTOMGJP19	查询职位流水号数量不能超过1000
XFTOMGJP20	查询职位编码长度不能超过8
XFTOMGJP21	查询职位名称长度不能超过150
