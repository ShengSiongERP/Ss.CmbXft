分页查询岗位
更新时间: 2025-04-17 07:30:00
接口调用地址
生产环境地址：https://api.cmbchina.com/hrm/hrm2/organization-management/openapi/position/query/page
测试环境地址：https://api.cmburl.cn:8065/xfthrm/hrm2uat/organization-management/openapi/position/query/page
接口描述
分页查询岗位
调用方式
METHOD: POST
CONTENT-TYPE: application/json
接口输入参数

请求体
仅必填全部展开
参数	必填	类型	最大长度	名称	示例值	字段说明
					
	否	ARRAY	1000	流水号列表	["0000**01"]	每项最大长度为10
sequenceNumbers
	否	STRING	150	岗位名称	产品*理	支持模糊查询
positionName
	否	ARRAY	1000	所属组织机构ID	["0**0","0**3"]	每项最大长度为4
organizationIds
	否	STRING	8	岗位编号	P**01	支持模糊查询
codeNumber
	是	LONG	-	当前页数	1	起始值，如1，默认为1
currentPage
	是	LONG	-	每页大小	50	起始值，如50，默认为10，最大支持1000
pageSize

currentPage
	是	LONG	-	每页大小	50	每页大小，如100，默认为10
pageSize

错误码说明
除HTTP返回码外，具体的业务异常错误码通常指的是返回报文中错误码字段的内容，常见通用错误码及解决方案请参考通用错误码。
错误码	错误信息
XFTOMGPS23	查询岗位流水号数量不能超过1000
XFTOMGPS24	查询岗位所属组织机构ID数量不能超过1000
XFTOMGPS25	查询岗位编码长度不能超过8
XFTOMGPS26	查询岗位名称长度不能超过150
